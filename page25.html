Program Pondok Pesantren

Pondok pesantren tahfidzul quran fathul ulum memiliki berbagai jenis program pembelajaran sebagai berikut  :

1

Program Unggulan

Berikut ini adalah beberapa program unggulan yang ada dipondok pesantren, sebagai berikut :
1). Sekolah Umum , Seperti :
      a. Raudhatul athfal
      b. Madrasah ibtidaiyah
      c. Madrasah tsanawiyah
      d. Madrasah aliyah kejuruan
      e. Usto paket setara mts/smpd
      f. Ulia setara dengan sma.
2). Sekolah Madrasah, seperti :
      a. ula
      b. wustho
      c. marhalah ula
3). sholat berjamaah
4). Sholat tahajjud
5). Haul majemuk
2

Program Kajian Kitab

Berikut ini adalah beberapa contoh kitab yang di pelajari dalam pondok pesantren, seperti :
1). Jurumiyah
2). Amtsilah at-tashrifiyah
3). Hidayatus shibyani
4). Adaabul 'alim wal muta'allim
5). Kitab-kitab kuning , seperti : 
      a. Ihya' ulumuddin
      b. Bidayatul hidayah
      c. Ta'lim muta'allim
      d. Dasyifatus saja’a
6). kitab-kitab dasar, seperti :
      a. Jurumiyah
      b. Amtsilah at-tashrifiyah
      c. Hidayatus shibyani
      d. Adaabul 'alim wal muta'allim
3

Program Alat Musik Islam 

Alat musik-musik tersebut sering digunakan di pondok pesantren untuk mengiringi kegiatan-kegiatan keagamaan seperti pembacaan Al-Qur'an, dzikir, shalawat, dan acara-acara keagamaan, seperti :
1). Alat musik gambus
2). Alat musik rebana
3). Alat musik marawis
4). Alat musik kendang
5). Alat musik kompang
4

Jadwal Program 

Berikut ini adalah jadwal program - program di pondok pesantren, sebagai berikut:
1. Sekolah umum : Jam 7:00 sampai jam 13:00
2. Sekolah madrasah : Jam 15:30 sampai jam 21:30
3. Sholat berjamaah : setiap waktu sholat tiba
4. Sholat tahajjut : Jam 03:30 
5. Haul majemuk : setiap tanggal 17 jumadil akhir
